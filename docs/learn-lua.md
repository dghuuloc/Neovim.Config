## Lua's Keywords
```
and       break     do        else      elseif    end
false     for       function  goto      if        in
local     nil       not       or        repeat    return
then      true      until     while
```

## Comments
```
-- This is a comment. It starts with two dashes
--[[ This is also
     a comment.
     But it spans multiple lines!
--]]
```
